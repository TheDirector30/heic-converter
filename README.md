# heic-converter

Free, private HEIC to JPG converter. Runs entirely in your browser. No uploads, no accounts.

## Features

- Drag-and-drop or click to select files
- Batch conversion — convert multiple HEIC/HEIF files at once
- Download individual JPGs or all files as a ZIP
- Progress tracking per file
- 100% client-side — nothing is ever uploaded to a server

## Browser Compatibility

| Browser | Support |
|---------|---------|
| Chrome / Edge | Full |
| Safari | Full |
| Firefox | Limited (HEIC support varies by OS) |

For best results, use Chrome or Safari.

## Usage

1. Open the app in your browser
2. Drag and drop HEIC files onto the drop zone, or click to select files
3. Wait for conversion to complete
4. Download individual JPGs or click "Download All as ZIP"

## Privacy

All conversions happen locally in your browser. No files are uploaded anywhere.
